# Agente de Ejemplo

Este es un agente simple de ejemplo para Claude Code.

## Propósito

Ayudar con tareas específicas de este proyecto.

## Instrucciones

Cuando este agente sea invocado:

1. Analiza el código del proyecto
2. Proporciona sugerencias de mejora
3. Ayuda a depurar problemas comunes

## Contexto del Proyecto

Este es un proyecto Python básico. Asegúrate de:
- Seguir PEP 8 para estilo de código
- Documentar funciones con docstrings
- Escribir tests cuando sea apropiado
